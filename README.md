3D Portfolio Website

This is a modern and interactive 3D developer portfolio built using Next.js, TypeScript, and Tailwind CSS. The portfolio features smooth animations, immersive 3D elements, and a clean responsive design to showcase projects, technical skills, and professional experience.

The goal of this project is to create a visually impressive portfolio that stands out to recruiters and clients while demonstrating modern frontend development skills.

Key Features:

Interactive 3D animations and geometric backgrounds

Smooth scrolling and modern UI/UX

Fully responsive design for all devices

Projects, skills, and experience sections

Contact section for collaboration

Built with modern React (Next.js App Router)

Optimized performance and clean code structure

Tech Stack:

Next.js

TypeScript

Tailwind CSS

React Three Fiber / 3D components

Framer Motion (animations)
